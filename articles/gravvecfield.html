Gravity Simulation and 2D Gravity Vector Field

Gravity is one of the fundamental forces of nature that attracts every object with mass to every other object with mass. To study the effects of gravity on different objects, we can use computer simulations that model the gravitational force and the motion of the objects under its influence.

One such simulation can be found here, which uses C++ and OpenGL to create a 2D gravity vector field and a gravity physics system. A vector field is a map of vectors that assigns a vector to each point in a plane or space. The vector represents the direction and magnitude of a certain quantity, such as force, velocity, or acceleration. A 2D gravity vector field shows how the gravitational force varies at different points in a plane.

The simulation uses a class called GravityPhysicsSystem that takes a parameter for the strength of gravity and has a method called update that takes a vector of objects, a time step, and a number of substeps. The update method loops through all pairs of objects and applies the attractive force between them using the formula:

F=Gm1m2r2F = \frac{Gm_1m_2}{r^2}

where F is the force, G is the gravitational constant, m1 and m2 are the masses of the objects, and r is the distance between them. The method also updates the position of each object based on its velocity using Euler’s method.

The simulation also uses a class called SveGameObject that represents an object with a 2D transform (translation, rotation, and scale) and a 2D rigid body (mass and velocity). The objects are rendered as colorful circles with different sizes depending on their mass. The simulation shows the 2D gravity vector field as a grid of arrows that point towards the center of mass of all the objects.

The simulation demonstrates some interesting phenomena related to gravity, such as orbital motion, gravitational slingshot, and gravitational collapse. The user can experiment with different scenarios by changing the number, mass, and velocity of the objects.

The simulation is a fun and educational way to explore gravity and its effects on different objects. It can also be used as a starting point for further development and improvement, such as adding more features, optimizing performance, or extending it to 3D.
